   ![ssm_01](https://yqfile.alicdn.com/dfede98114164bd5ce5aca1bef973b291cdbe423.png)
# 第二步： 点击project structure,点击Project，选择自己的jdk版本
   ![ssm_02](https://yqfile.alicdn.com/1f76c9f197bbc6ea15a5bcc8fd6530ad3c33b19b.png)
# 第三步： 点击Model ，点击+ 号，选择Web
   ![ssm_03](https://yqfile.alicdn.com/eda114ef3284ff4ea8f42fc28b9cd63b80b50d5b.png)
# 第四步： 点击libraries ,点击 + 号，选择自己的项目。 并在右边部分添加自己项目依赖的包。
   ![ssm04](https://yqfile.alicdn.com/7babafec5ab31464dd19fd7405cab8e1e641b7a5.png)
# 第五步： 点击Facets，点击+ 号，选择Web,
   ![ssm_05](https://yqfile.alicdn.com/31675010a8f374dc1f5e7973c4f8fb9cfe888232.png)
# 第六步： 点击AriFates，点击 + 号，选择 Web Application Explored, 选择自己的项目。
   ![ssm06](https://yqfile.alicdn.com/f73c9f28daaafe0f1a25e600d2fc5a444ce1c5a8.png)
